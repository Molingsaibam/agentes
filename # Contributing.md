# Contributing

Padrões básicos para contribuir com este repositório.

- Faça forks/branches para features.
- Commits atômicos e descritivos.
- Atualize CHANGELOG.md para mudanças relevantes.

Modelo de commit:

- feat(agent): descrição curta
- fix(agent): correção breve
- docs: documentação
- chore: manutenção

Pull requests:
- Explique o que mudou e por quê.
- Vincule issues quando aplicável.
